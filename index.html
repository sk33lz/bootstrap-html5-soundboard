Flash is Dead!

Long Live Soundboards!

The Bootstrap HTML5 Soundboard is an HTML5 Soundboard proof of concept. It doesn't require Flash and runs on modern web technologies like HTML5, jQuery, and CSS3. It is built on top of Boostrap, so it should run on any modern web browser and any mobile devices out of the box.

Click on an image to play the audio clip compatible with your device.
